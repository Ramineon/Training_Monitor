# Training Monitor

A simple real-time training dashboard built with **Next.js** and React.

## Features
- Live Epoch counter
- Decreasing Loss value (simulating model training)
- Clean dark UI with Tailwind CSS
- Client-side interactivity using `useState`

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React
