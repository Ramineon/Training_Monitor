# Training Monitor

a UI sandbox for practicing React state, built around a simulated training loop.

## Features
- Live Epoch counter
- Decreasing Loss value (simulating model training)
- Clean dark UI with Tailwind CSS
- Client-side interactivity using `useState`

## Tech Stack
- Next.js (App Router)
- TypeScript
- Tailwind CSS
- React
